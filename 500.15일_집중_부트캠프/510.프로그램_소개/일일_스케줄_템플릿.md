## 🕐 시간대별 최적 스케줄

### 1. 새벽 학습 (04:00-07:00)

```yaml
장점:
  - 방해 요소 제로 (세상이 조용)
  - 뇌 최고 컨디션 (수면 직후 집중력 최고)
  - 성취감 조기 획득 (하루 시작 전 완료)
  - 저녁 시간 자유

단점:
  - 일찍 일어나기 힘듦
  - 전날 일찍 자야 함 (22:00 취침)
  - 졸음과의 싸움 (초기)

적합한 사람:
  - 아침형 인간
  - 저녁에 에너지 바닥나는 사람
  - 조용한 환경이 필수인 사람

준비 사항:
  - D-3일부터 기상 시간 연습 (05:30 → 05:15 → 05:00)
  - 전날 밤 학습 자료 미리 세팅
  - 알람 3개 설정 (끄고 다시 자는 것 방지)
```

### 2. 오전 학습 (09:00-12:00)

```yaml
장점:
  - 출근 후 컨디션 안정
  - 점심 전 집중력 좋음
  - 직장인/목회자 맞춤형

단점:
  - 업무 시간 조정 필요
  - 회의/전화 방해 가능성
  - 상사/동료 눈치

적합한 사람:
  - 업무 시간 조정 가능한 목회자/간사
  - 재택근무 가능한 직장인
  - 자영업자

준비 사항:
  - 상사에게 사전 협의 (교육 시간 확보)
  - 회의실 예약
  - "교회 시스템 구축 업무" 명목
  - 전화/이메일 알림 끄기
```

### 3. 오후 학습 (14:00-17:00)

```yaml
장점:
  - 점심 후 소화되면 집중력 회복
  - 저녁 시간 자유
  - 업무 중 분할 가능

단점:
  - 점심 직후 졸음
  - 오전 업무 피로 누적
  - 저녁 약속 영향 없음

적합한 사람:
  - 오후형 인간
  - 점심 시간 길게 확보 가능한 사람 (2-3시간)
  - 저녁 사역/가족 시간 필수인 사람

준비 사항:
  - 점심 과식 금지 (졸음 유발)
  - 커피 한 잔 (30분 전)
  - 15분 power nap (선택)
```

### 4. 저녁 학습 (20:00-23:00) ⭐ 가장 대중적

```yaml
장점:
  - 하루 업무 종료 후 집중
  - 가족 시간 후 개인 시간
  - 마감 효과 (자기 전까지 완료 동기)
  - 대부분 사람에게 적합

단점:
  - 하루 피로 누적
  - 가족 방해 가능성
  - 늦게 끝나면 수면 부족

적합한 사람:
  - 저녁에 집중력 최고조인 사람
  - 아침 일찍 못 일어나는 사람
  - 저녁 시간 통제 가능한 사람

준비 사항:
  - 가족에게 사전 공지 (방해 금지 시간)
  - 저녁 식사 가볍게 (18:00-19:00)
  - 학습 공간 미리 정리 (19:30)
  - 전날 23:00 취침 (수면 6-7시간 확보)
```

### 5. 심야 학습 (23:00-02:00) ⚠️ 비추천

```yaml
장점:
  - 완전한 조용함
  - 방해 요소 제로
  - 올빼미족 적합

단점:
  - 수면 부족 (치명적)
  - 다음 날 컨디션 최악
  - 장기적 건강 악영향
  - 학습 효율 낮음 (피로)

권장:
  ❌ 비추천!
  - 15일 지속 불가능
  - 번아웃 위험
  - 건강 악화

대안:
  - 새벽형으로 전환 (22:00 취침 → 05:00 기상)
  - 또는 저녁 20:00-23:00
```

```yaml

  직장: 교회 (주 40시간)
  가족: 미혼, 혼자 거주
  선택: 아침형 스케줄

05:00 기상
05:15-08:00 부트캠프 학습 (2.5시간)

09:00 교회 도착
09:00-18:00 교회 업무
18:00-19:00 저녁 식사
19:00-21:00 개인 시간 (운동, TV, 휴식)
21:00-22:00 복습 + 내일 준비 (1시간)
22:00 취침 (7시간 수면)

총 학습 시간: 하루 3.5시간

  ✅ 고정된 시간 (매일 05:15 시작)

  - 15일 완벽 출석
```
