## 로고 시스템

### 로고 컨셉

```
핵심 모티브: 나무 (Tree of Growth)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

       🌳
      /│\        ← 가지/잎 = 리더들 (Leadership)
     / | \
    /  |  \
   ┌───┴───┐    ← 줄기 = 교회 공동체
   │       │
   │       │
   └───┬───┘
      ╱│╲       ← 뿌리 = 시스템 (System)
     ╱ │ ╲
    ╱  │  ╲

상징 의미:
• 뿌리 (Root) = CHEONGPA 운영 시스템
  → 보이지 않지만 견고한 기초

• 줄기 (Trunk) = 교회 공동체
  → 영양분을 전달하는 통로

• 가지/잎 (Branches) = Bluehill21Leader
  → 성장하고 열매 맺는 리더들

• 전체 = 건강하게 번영하는 교회
```

### 컬러 사용 비율

```
이상적인 컬러 비율 (60-30-10 Rule):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

60% - Neutral (White, Light Gray)
      → 배경, 여백

30% - Primary (Deep Blue)
      → 텍스트, 헤더

10% - Accent (Forest Green, Gold)
      → 강조, CTA

예시:
┌────────────────────────────────────┐
│ [White Background - 60%]           │
│                                    │
│  Deep Blue Title - 30%             │
│  Deep Blue Paragraph Text          │
│                                    │
│  [Forest Green Button - 10%] ←─── Accent!
│                                    │
└────────────────────────────────────┘
```

### 컬러 접근성 (Accessibility)

```
WCAG 2.1 AA 기준 준수:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• Deep Blue (#1A365D) on White (#FFFFFF)
  → 대비율 12.5:1 (AAA)

• Dark Gray (#2D3748) on White (#FFFFFF)
  → 대비율 11.4:1 (AAA)

• Forest Green (#228B22) on White (#FFFFFF)
  → 대비율 4.8:1 (AA)

❌ Poor Contrast (피할 것):
• Light Gray (#F7FAFC) on White (#FFFFFF)
  → 대비율 1.1:1 (실패)

## 금지 사항

### 로고 사용 금지

```
❌ 1. 비율 변형
   ┌────┐
   │🌳  │  ← 세로로 늘림
   └────┘

❌ 2. 회전
     🌳  ← 45도 회전
    ╱ ╲

❌ 3. 왜곡
   ╱🌳╲  ← 기울임

❌ 4. 색상 변경 (브랜드 컬러 외)
   🌳 (Pink) ← 핑크색

❌ 5. 외곽선 추가
   ┌─🌳─┐  ← 테두리

❌ 6. 그림자 과도
   🌳
   ▓▓▓  ← 진한 그림자

❌ 7. 다른 요소와 겹침
   🌳[Text]  ← 텍스트 겹침

❌ 8. 배경 대비 부족
   [Gray]🌳[Gray]  ← 회색 배경에 회색 로고
```

### 컬러 사용 금지

```
❌ 1. 브랜드 컬러 외 색상
   네온 컬러, 형광 컬러

❌ 2. 과도한 그라데이션
   6가지 이상 색상 그라데이션

❌ 3. 대비 부족
   회색 텍스트 on 흰 배경 (3:1 미만)

❌ 4. RGB-CMYK 불일치
   웹/인쇄 색상 차이 방치
```

### 타이포그래피 금지

```
❌ 1. 비공식 폰트 사용
   Comic Sans, Papyrus 등

❌ 2. 과도한 장식
   그림자, 외곽선, 엠보싱 동시 사용

❌ 3. 전체 대문자 남용
   긴 문장을 ALL CAPS

❌ 4. 행간 부족
   Line Height 1.0 이하

❌ 5. 폰트 크기 과다
   한 화면에 10가지 사이즈

❌ 6. 정렬 불일치
   좌/중/우 정렬 혼재
```

### 메시지 금지

```
❌ 1. 과장된 주장
   "100% 성장 보장"
   "기적의 솔루션"

❌ 2. 부정적 표현
   "망해가는 교회"
   "실패한 목회"

❌ 3. 비교/비난
   "다른 컨설팅은 XX"
   "OO교회는 안 됨"

❌ 4. 전문 용어 남발
   "KPI 최적화 via ML"

❌ 5. 종교적 강요
   "믿지 않으면..."
   "하나님의 심판"

❌ 6. 정치적 입장
   특정 정당/이념 지지
```

### 이미지 사용 금지

```
❌ 1. 저작권 침해
   출처 불명 이미지
   무단 사용

❌ 2. 부적절한 이미지
   폭력, 선정성
   차별적 표현

❌ 3. 저품질 이미지
   해상도 72dpi 이하
   픽셀 깨짐

❌ 4. 과도한 편집
   필터 10개 중첩
   인위적인 느낌

❌ 5. 맥락 부적합
   교회와 무관한 이미지
   (예: 나이트클럽 사진)
