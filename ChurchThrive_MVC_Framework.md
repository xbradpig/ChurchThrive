## MISSION STATEMENT (사명)

### 우리의 사명

```
ChurchThrive는
교회의 생명징후를 정확히 진단하고,
시스템과 리더십의 두 날개로
모든 교회가 하나님 나라의 비전을
온전히 실현하도록 돕습니다.

We diagnose the vital signs of churches
and empower them with robust systems and
transformative leadership to fully realize
God's Kingdom vision.
```

### 사명 분해 (Mission Breakdown)

#### 1. WHO (누구를 위한가)
**모든 교회 - Every Church**

- ✅ **규모 무관**: 개척 교회(20명) ~ 대형 교회(1000명+)
- ✅ **교단 무관**: 장로교, 감리교, 순복음, 침례교, 비교단
- ✅ **지역 무관**: 서울, 지방, 농어촌, 해외 한인 교회
- ✅ **문화 무관**: 전통적 교회, 현대적 교회, 하이브리드

**구체적 타겟:**
```
Primary Target (1차):
- 출석 50-300명 중소형 교회
- 성장 정체기 교회
- 세대 교체기 교회
- 시스템 부재로 고통받는 교회

Secondary Target (2차):
- 개척 1-3년 교회 (예방 차원)
- 부흥기 대형 교회 (고도화)
- 교회 연합/연합회
```

#### 2. WHAT (무엇을 하는가)
**진단 → 솔루션 → 성장**

##### Phase 1: 진단 (Diagnosis)
```
교회의 7가지 생명징후 측정:

1. 비전/사명 명확성      Vision Clarity
2. 리더십 역량           Leadership Capacity
3. 운영 시스템           Operational System
4. 재정 건전성           Financial Health
5. 교육/양육 체계        Discipleship System
6. 선교/전도 활동        Mission & Outreach
7. 공동체 건강도         Community Vitality

측정 도구:
- Leadership Vitals Index (LVI)
- 7-Dimension Church Health Matrix
- 50-Question Assessment
- Data-Driven Dashboard
```

##### Phase 2: 솔루션 (Solutions)
```
Two-Pillar Approach (두 기둥 접근법):

┌─────────────────────────────────────────────┐
│  Pillar 1: CHEONGPA System                  │
│  체계적인 교회 운영 시스템                    │
├─────────────────────────────────────────────┤
│  ✓ 회의록 표준화                             │
│  ✓ 예배/설교 관리 체계                       │
│  ✓ 교인 정보 관리 시스템                     │
│  ✓ 재정 투명성 확보                          │
│  ✓ 통계/보고 자동화                          │
│  ✓ 부서 간 협업 체계                         │
│  ✓ 지식 자산 축적 (10년 활용 가능)           │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│  Pillar 2: Bluehill21Leader Program         │
│  변혁적 리더십 개발 시스템                    │
├─────────────────────────────────────────────┤
│  ✓ 6대 핵심 역량 개발                        │
│    • 영적 리더십                             │
│    • 섬김의 리더십                           │
│    • 비전과 사명                             │
│    • 의사소통                                │
│    • 팀빌딩                                  │
│    • 갈등관리                                │
│  ✓ 멘토링/코칭 시스템                        │
│  ✓ 프로젝트 기반 학습                        │
│  ✓ 사전/사후 평가                            │
│  ✓ 차세대 리더 파이프라인                    │
└─────────────────────────────────────────────┘

→ System Without Leaders = 영혼 없는 기계
→ Leaders Without System = 방향 잃은 열정
→ System + Leaders = Thriving Church ✅
```

##### Phase 3: 성장 (Growth)
```
Measurable Outcomes (측정 가능한 결과):

정량적 성장:
• 출석 인원 증가: 평균 15-20% (1년)
• 새가족 정착률: 60% → 80%
• 재정 건전성: 예산 대비 집행률 85-95%
• 사역 참여율: 30% → 50%

정성적 성장:
• 비전 공유도: "우리 교회가 어디로 가는지 안다"
• 리더 역량: 6대 역량 평균 3.5 → 4.5 (5점 척도)
• 공동체 건강: 갈등 감소, 관계 개선
• 영적 성숙: 제자 배출, 선교 참여 증가
```

#### 3. HOW (어떻게 하는가)
**데이터 기반 + 성경적 원칙 + 맞춤형 솔루션**

##### 방법론 1: 과학적 진단
```
1. 사전 설문 (온라인)
   - 7가지 영역 50개 질문
   - 담임목사, 리더, 평신도 각각 응답
   - 자동 분석 → LVI 지수 산출

2. 현장 인터뷰 (2-4시간)
   - 담임목사 심층 인터뷰 (1시간)
   - 핵심 리더 그룹 인터뷰 (1시간)
   - 평신도 포커스 그룹 (1시간)
   - 현장 관찰 (예배, 공간, 시스템)

3. 데이터 분석
   - 출석/헌금 추이 (3-5년)
   - 연령별/세대별 분포
   - 부서별 활동 지표
   - 재정 구조 분석

4. 종합 진단 리포트
   - 7가지 영역별 점수
   - 강점/약점 분석
   - 3가지 핵심 이슈 도출
   - 우선순위 솔루션 제안
```

##### 방법론 2: 맞춤형 솔루션
```
One Size Does NOT Fit All

교회별 맞춤 설계:
• 규모: 소형/중형/대형
• 교단: 장로교/감리교/순복음 등
• 상황: 개척기/성장기/성숙기/위기
• 문제: 비전 부재/리더 부족/시스템 문제/재정 위기

3개월 실행 계획:
Week 1-4:   Quick Wins (빠른 성과)
Week 5-8:   Core Systems (핵심 시스템)
Week 9-12:  Leadership (리더 개발)
```

##### 방법론 3: 지속 성장 지원
```
After-Care System:

1개월 후: 첫 번째 체크인
3개월 후: 중간 평가
6개월 후: 재진단 (LVI 재측정)
12개월 후: 성과 측정 + 성공 사례화

지속 지원:
• 월간 성장 리포트
• 이메일/전화 상담
• 온라인 커뮤니티 (동료 교회와 교류)
• 연간 리더십 컨퍼런스
```

#### 4. WHY (왜 하는가)
**하나님 나라 확장**

```
궁극적 목적 (Ultimate Purpose):

하나님의 영광을 위해 (Soli Deo Gloria)
교회가 본연의 사명을 회복하여
세상을 변화시키는 빛과 소금이 되게 함

"그런즉 너희가 먹든지 마시든지
무엇을 하든지 다 하나님의 영광을 위하여 하라"
(고린도전서 10:31)
```

**우리가 믿는 것 (What We Believe):**

1. **교회는 하나님의 걸작품이다**
   - 예수님이 세우신 교회 (마 16:18)
   - 그리스도의 몸 (엡 1:22-23)
   - 세상의 빛과 소금 (마 5:13-16)

2. **모든 교회는 성장할 수 있다**
   - 크기가 아닌 건강이 우선
   - 건강한 교회는 자연스럽게 성장
   - 성장 = 숫자 + 영적 깊이 + 영향력

3. **리더가 바뀌면 교회가 바뀐다**
   - 목회자의 역량이 교회의 천장
   - 평신도 리더 양성이 성장의 열쇠
   - "리더를 세우는 리더"가 필요

4. **시스템이 문화를 만든다**
   - 좋은 의도만으로는 부족
   - 반복 가능한 시스템이 필수
   - 시스템이 있어야 성장이 지속

## CORE VALUES (핵심 가치)

### 우리의 7가지 핵심 가치

```
ChurchThrive의 모든 의사결정과 행동의 기준

1. Kingdom First (하나님 나라 우선)
2. Excellence (탁월함)
3. Integrity (진실성)
4. Partnership (동반자십)
5. Innovation (혁신)
6. Empowerment (역량 강화)
7. Sustainability (지속 가능성)
```

### Core Value 1: Kingdom First (하나님 나라 우선)

**정의:**
```
우리는 교회의 성장이 아닌
하나님 나라의 확장을 목표로 합니다.
모든 전략, 솔루션, 성과는
"하나님 나라에 도움이 되는가?"로 평가됩니다.
```

**성경적 근거:**
```
"먼저 그의 나라와 그의 의를 구하라
그리하면 이 모든 것을 너희에게 더하시리라"
(마태복음 6:33)
```

**실천 사항:**
- ✅ 교회 간 경쟁이 아닌 협력 추구
- ✅ 단기 성과보다 장기 건강 우선
- ✅ 숫자보다 영적 깊이 강조
- ✅ 개교회주의 극복, 연합 사역 지향
- ✅ 수익보다 사명 우선 (비영리 정신)

**측정 지표:**
```
• 교회 간 협력 프로젝트 수
• 선교 재정 증가율
• 지역사회 섬김 프로그램 수
• 영적 성숙도 점수 증가
```

---

### Core Value 2: Excellence (탁월함)

**정의:**
```
하나님께 드리는 사역은
"대충"이나 "적당히"가 아닌
최선과 탁월함으로 이루어져야 합니다.
우리는 교회 컨설팅의 새로운 기준을 만듭니다.
```

**성경적 근거:**
```
"무릇 네 손이 일을 당하는 대로 힘을 다하여 할지어다"
(전도서 9:10)

"지혜 있는 자의 마음은 그의 입을 슬기롭게 하고
그의 입술에 지식을 더하느니라"
(잠언 16:23)
```

**실천 사항:**
- ✅ 업계 최고 수준의 진단 도구 개발
- ✅ 지속적인 R&D 투자 (매출의 20%)
- ✅ 컨설턴트 전문성 지속 교육
- ✅ 100% 만족도가 아니면 환불
- ✅ 모든 자료 3번 검토 후 출시

**품질 기준:**
```
• 진단 정확도: 95% 이상

1. 사도의 가르침 (Teaching)      → 교육 체계

3. 떡을 뗌 (Worship)              → 예배 생명력
4. 기도 (Prayer)                  → 영적 능력

→ ChurchThrive의 진단 7가지 영역과 연결
```

```
"너희 중에 누구든지 크고자 하는 자는

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• 권위가 아닌 영향력
• 지배가 아닌 돌봄

→ BH21L 핵심 역량 중 "섬김의 리더십"
```

"그러므로 누구든지 나의 이 말을 듣고 행하는 자는

→ CHEONGPA 시스템의 근거
```

#### 4. 성장의 원리: 씨앗과 열매

```
"내가 심었고 아볼로는 물을 주었으되
오직 하나님께서 자라나게 하셨나니"
