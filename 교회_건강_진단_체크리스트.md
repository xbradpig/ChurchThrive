## 진단 개요

### Leadership Vitals Index (LVI)란?

```
LVI = 교회의 '생명징후'를 측정하는 ChurchThrive 독자 지표

의료 검진처럼 교회의 건강 상태를
7가지 핵심 영역으로 과학적으로 측정하여
개선이 필요한 부분을 정확히 찾아냅니다.

┌────────────────────────────────────────┐
│  의료 검진        →    교회 진단       │
├────────────────────────────────────────┤
│  혈압, 혈당       →    7가지 생명징후   │
│  정상 범위        →    LVI 지수        │
│  처방전           →    맞춤 솔루션      │
│  정기 검진        →    분기별 재진단    │
└────────────────────────────────────────┘
```

### 7가지 영역 (Seven Vital Signs)

```
1. 비전/사명 명확성 (Vision & Mission Clarity)
   → 교회가 어디로 가는지 알고 있는가?

2. 리더십 역량 (Leadership Capacity)
   → 리더들이 준비되어 있는가?

3. 운영 시스템 (Operational System)
   → 체계적으로 운영되고 있는가?

4. 재정 건전성 (Financial Health)
   → 재정이 투명하고 건강한가?

5. 교육/양육 체계 (Discipleship System)
   → 성도들이 성장하고 있는가?

6. 선교/전도 활동 (Mission & Outreach)
   → 교회 밖을 향하고 있는가?

7. 공동체 건강도 (Community Vitality)
   → 관계가 건강한가?
```

### 진단 대상

```
필수 응답자:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ 담임목사/담임전도사
✅ 핵심 리더 (장로, 권사, 집사 등 3-5명)
✅ 평신도 대표 (일반 성도 3-5명)

→ 총 7-11명 응답 권장
→ 다양한 관점에서 객관적 진단

선택 응답자:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• 부목사/전도사
• 각 부서 팀장
• 청년부 대표
• 새가족 대표

→ 규모가 클수록 더 많은 응답자
```

### 소요 시간

```
응답 시간: 약 15-20분
분석 시간: 즉시 (자동 계산)
리포트:   응답 완료 후 24시간 내 발송
```

## 응답 방법

### 점수 기준

```
5점 척도 (Likert Scale):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

5점: 매우 그렇다 (Strongly Agree)
     → 이상적인 상태, 문제 없음

4점: 그렇다 (Agree)
     → 양호한 상태, 약간 개선 여지

3점: 보통이다 (Neutral)
     → 평범한 상태, 명확한 개선 필요

2점: 아니다 (Disagree)
     → 문제가 있는 상태, 즉시 개선 필요

1점: 전혀 아니다 (Strongly Disagree)
     → 심각한 상태, 긴급 조치 필요

N/A: 해당 없음
     → 교회 상황상 적용 불가
```

### 응답 가이드

```
✅ DO (이렇게 하세요):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• 솔직하게 응답 (방어적 태도 ❌)
• 현재 상태 기준 (이상이 아닌 현실)
• 첫 인상대로 응답 (과도한 고민 ❌)
• 구체적 사례 떠올리기
• 평균적 상황 기준 (예외 아닌 일반)

❌ DON'T (피하세요):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• "좋게 보이려고" 높은 점수
• "너무 비관적으로" 낮은 점수
• 모든 질문에 3점 (중간)
• 담임목사 눈치 보며 응답
• 빠르게 대충 응답
```

### 응답 예시

```
질문: "우리 교회의 비전이 명확하게 정의되어 있다"

5점: ✅ 명문화된 비전문이 있고, 모든 성도가 암송
4점: ✅ 비전문은 있으나, 일부만 알고 있음
3점: ⚠️ 비전이 있긴 한데, 막연함
2점: ❌ 비전이 있는지 없는지 모르겠음
1점: ❌ 비전이 전혀 없음
```

### 영역 4: 재정 건전성 (Financial Health)

### 영역별 점수 계산

```
각 영역 점수 = (영역 내 질문 합계) ÷ (응답 질문 수)

예시:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
영역 1 (비전/사명):
Q1: 4점
Q2: 3점
Q3: 5점
Q4: 4점
Q5: 3점
Q6: 4점
Q7: N/A (제외)

→ 영역 1 점수 = (4+3+5+4+3+4) ÷ 6 = 3.83점
```

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
영역 1: 3.83
영역 2: 3.50
영역 3: 3.14
영역 4: 4.00
영역 5: 3.43
영역 6: 2.86
영역 7: 4.14

→ LVI = 24.90 ÷ 7 = 3.56점
```
