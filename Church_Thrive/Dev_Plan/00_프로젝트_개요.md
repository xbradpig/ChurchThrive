# 00. 프로젝트 개요

## 서비스 정보

- **서비스명**: ChurchThrive App (교회성장연구소 앱)
- **서비스 유형**:  교회 혁신 성장 통합  SaaS 플랫폼 (Software as a Service)
- **타겟 시장**: 한국 개신교 교회 (중형교회 50-300명 중심)
- **플랫폼**: PWA (웹) + React Native (iOS/Android) 동시 개발

---

## 프로젝트 목표

교인관리, 교회 행정관리, 말씀노트(녹음→STT + 텍스트), 훈련관리, 사역관리를 **하나의 플랫폼에 통합**하여 교회의 행정 효율화와 영적 성장을 동시에 지원하는 **교회 총괄 관리 앱**을 구축한다.

### 핵심 차별화 요소

1. **AI 기반 한국어 설교 STT**: 설교 음성을 텍스트로 자동 변환 (Google Cloud STT + Whisper + CLOVA 3중화)
2. **영적성장 도구 통합**: 말씀노트 + 말씀 암송(다국어) + 말씀 통독/필사를 하나의 앱에서 제공
3. **ChurchThrive 컨설팅 생태계 연동**: 기존 127명 부트캠프 졸업생 기반 + LVI 교회건강진단 통합
4. **교회 홈페이지 자동 빌드**: 가입 시 AI가 설문을 분석하여 맞춤형 홈페이지 무료 생성
5. **비용 최소화**: Cloudflare Pages + Supabase 무료 티어 기반으로 교회 부담 최소화

---

## 비전 & 미션

### 비전
> 모든 교회가 건강하게 성장할 수 있도록 기술로 돕는다

### 미션
> 행정의 디지털 전환과 영적 성장 도구를 통합하여, 목회자와 성도가 본질에 집중할 수 있는 환경을 만든다

### 핵심 가치
1. **교회 중심 (Church-First)**: 교회의 실제 필요에 맞춘 기능 설계
2. **접근성 (Accessibility)**: 디지털 리터러시가 낮은 고령 교인도 사용 가능한 UX
3. **데이터 주권 (Data Sovereignty)**: 교회가 데이터를 소유하고 완전히 통제
4. **비용 효율 (Cost Efficiency)**: 무료 티어로 시작, 성장에 따른 합리적 과금
5. **보안 우선 (Security First)**: 교인 개인정보 보호법(PIPA) 완전 준수

## 프로젝트 범위

### Phase 1 - MVP (Month 1-4)
- 가입/인증/교회 격리 시스템
- 교인관리 (교적, 출석, 심방, QR 등록)
- 말씀노트 (녹음 + 텍스트, 오프라인 지원)
- 교회 행정관리 기본 (공지/알림, 조직도, 직분관리, 권한위임)
- 교회 홈페이지 자동 빌드
- 구역/셀 관리

### Phase 2 - Growth (Month 5-7)
- 한국어 STT (음성→텍스트 자동 변환)
- 전자결재 시스템
- 재정관리, 주보 생성, 행사 관리
- 훈련 프로젝트 관리
- 말씀 암송 (다국어 + TTS)
- 말씀 통독 + 필사
- 구독 결제 연동

### Phase 3 - Maturity (Month 8-10)
- LVI 교회 건강 진단
- 봉사 스케줄링/로테이션
- 회의록 + 액션아이템 추적
- 캘린더/SMS 연동

### Phase 4 - Enterprise (Month 11-16)
- 멀티캠퍼스 지원
- AI 추천 (LVI 기반)
- API 공개, 화이트라벨
- 영어 지원 (재외동포)

## 프로젝트 일정 개요

```
Month 1-2  : 기반 구축 (인증, DB, 교인관리 기본)
Month 3-4  : MVP 완성 (말씀노트, 행정, 홈페이지빌더)
Month 5    : 베타 출시 (파일럿 교회 3-5개)

Month 8-10 : Phase 3 확장 기능
Month 11-16: Phase 4 엔터프라이즈 기능
Month 28-30: 손익분기점 도달 예상
```
