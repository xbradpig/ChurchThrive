## 디자인 원칙 요약

1. **모바일 우선 (Mobile First)** - 모바일에서 시작하여 태블릿/데스크톱으로 확장
2. **오프라인 우선 (Offline First)** - 핵심 기능(녹음, 노트, 출석)은 오프라인에서 완전 동작
3. **3탭 규칙** - 모든 주요 기능에 3탭 이내로 접근 가능
4. **고령 친화** - 최소 16px 폰트, 44x44px 터치 타겟, 간결한 UI
5. **따뜻함과 신뢰** - 교회 맥락에 맞는 감성적 디자인
