## ChurchThrive 소개

### 우리는 누구인가?

```
┌─────────────────────────────────────────┐
│  ChurchThrive 교회성장연구소             │
├─────────────────────────────────────────┤
│                                         │
│  교회의 생명징후를 정확히 진단하고       │
│  시스템과 리더십의 두 날개로             │
│  모든 교회가 하나님 나라의 비전을        │
│  온전히 실현하도록 돕는                  │
│  교회 성장 컨설팅 전문 기관              │
│                                         │
└─────────────────────────────────────────┘

창립: 2025년
본사: 서울특별시
팀원: 10명 (2025년 목표)
섬기는 교회: 50개 → 500개 (2030년)
```

### 우리가 하는 일

```
1. 교회 건강 진단 (Diagnosis)
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   • LVI (Leadership Vitals Index) 측정
   • 7가지 생명징후 평가
   • 50개 질문 종합 진단
   • 데이터 기반 분석

2. 맞춤형 솔루션 제공 (Solutions)
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   A. CHEONGPA System (운영 시스템)
      • 회의록 표준화
      • 통계/보고 자동화
      • 예배/설교 관리
      • 교인/재정 관리

   B. Bluehill21Leader (리더십 개발)
      • 6대 핵심 역량 교육
      • 멘토링/코칭 시스템
      • 프로젝트 기반 학습
      • 사전/사후 평가

3. 지속 성장 지원 (Growth)
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   • 월간 성장 리포트
   • 분기 재진단
   • 온라인 커뮤니티
   • 연간 리더십 컨퍼런스
```

### 우리의 차별점

vs. 일반 경영 컨설팅:

✅ 데이터 기반 진단 (느낌 ❌)

## MVC (Mission, Vision, Core Values)

### 🎯 MISSION (사명)

```
교회의 생명징후를 정확히 진단하고,
시스템과 리더십의 두 날개로
모든 교회가 하나님 나라의 비전을
온전히 실현하도록 돕습니다.
```

**당신의 역할:**
- 모든 업무는 이 사명을 실현하기 위함
- "이 일이 교회 성장에 도움이 되나?"를 항상 질문
- 단기 성과보다 장기 건강 추구

### 🔭 VISION (비전)

```
2030년까지
500개 교회의 생명징후 회복
5,000명 차세대 리더 양성
대한민국 교회를 세계 선교의 허브로
```

**우리의 여정:**
```
2025:  50교회 (Year 1 - 당신이 지금 여기!)
2026: 150교회 (누적)
2027: 300교회 (누적)
2030: 500교회 ✅

→ 당신은 이 역사적 여정의 핵심 멤버입니다!
```

### 💎 CORE VALUES (7대 핵심 가치)

#### 1. Kingdom First (하나님 나라 우선)

```
의미: 교회 성장이 아닌 하나님 나라 확장이 목표

실천:
• 교회 간 경쟁 부추기지 않기
• 단기 수익보다 사명 우선
• 연합 사역 적극 지지

당신의 일상:
"이 제안이 개교회만 키우는 건가,
 하나님 나라를 확장하는 건가?"
```

#### 2. Excellence (탁월함)

```
의미: 하나님께 드리는 사역은 최선으로

실천:
• 100% 만족도가 아니면 환불
• 자료 3번 검토 후 출시
• 업계 최고 수준 추구

당신의 일상:
"이 보고서, 내 이름 걸고 낼 수 있나?"
"더 좋게 할 방법은 없을까?"
```

#### 3. Integrity (진실성)

```
의미: 듣기 좋은 말이 아닌 필요한 진실

실천:
• 과장 광고 금지
• 비판적 피드백도 전달
• 실패 사례 공개

당신의 일상:
"목사님, 솔직히 말씀드리면..."
"이 부분은 개선이 필요합니다"
```

#### 4. Partnership (동반자십)

```
의미: 갑-을이 아닌 동역자 관계

실천:
• 교회와 정기 기도 모임
• 성공은 교회 공로, 실패는 우리 책임
• 계약 후에도 관계 유지

당신의 일상:
"우리가 함께 만들어가요"
"저도 배우고 있습니다"
```

#### 5. Innovation (혁신)

```
의미: "더 좋은 방법"을 끊임없이 추구

실천:
• 분기별 신규 솔루션
• AI/기술 적극 활용
• 실패 허용 문화

당신의 일상:
"이거 자동화할 수 있지 않을까?"
"다른 업계는 어떻게 하지?"
```

#### 6. Empowerment (역량 강화)

```
의미: 의존이 아닌 독립을 목표로

실천:
• 물고기 주기 ❌, 잡는 법 가르치기 ✅
• 교회가 스스로 운영하도록
• 6개월 후 독립 목표

당신의 일상:
"제가 대신 할게요" ❌
"함께 해보시겠어요?" ✅
```

#### 7. Sustainability (지속 가능성)

```
의미: 일회성 ❌, 10년 후에도 작동 ✅

실천:
• 시스템으로 전환
• 지식 문서화
• 세대 계승 준비

당신의 일상:
"이게 매뉴얼로 정리돼 있나?"
"목사님 바뀌어도 작동할까?"
```

## FAQ

### Q1. 교회 배경이 없는데 괜찮을까요?

```
A: 완전히 괜찮습니다! 🙆

이유:
• 객관적 시각이 오히려 장점
• 교회 지식은 교육으로 채움
• 컨설팅/분석 스킬이 더 중요
• 다양한 배경이 팀에 도움

제공:
• 신학 기본 교육 (4주)
• 교회 탐방 프로그램
• 멘토링 집중 지원
```

### Q2. 실수하면 어떻게 되나요?

```
A: 배우는 기회입니다! 💪

문화:
• "실수 OK, 숨김 NO"
• Fast Fail, Learn Fast
• 팀이 함께 커버

단,
• 같은 실수 반복 ❌
• 고의/태만 ❌
• 보고 안 함 ❌

→ 솔직하게 공유하고 함께 해결
```

### Q3. 성장 기회가 있나요?

```
A: 당연합니다! 🚀

커리어 패스:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
주니어 컨설턴트 (입사)
    ↓ (1-2년)
컨설턴트
    ↓ (2-3년)
시니어 컨설턴트
    ↓ (3-5년)
컨설팅 디렉터

or

교육 트랙으로 전환
운영 트랙으로 전환

지원:
• 연간 교육비 지원 (200만원)
• 컨퍼런스 참가 지원
• 자격증 취득 지원
• 1:1 코칭
```

### Q4. 급여 체계는 어떻게 되나요?

```
A: 투명하고 공정합니다! 💰

원칙:
• 직급별 기본급 (공개)
• 성과급 (분기별)
• 인센티브 (목표 달성 시)

복리후생:
• 4대 보험
• 퇴직금
• 명절 상여금
• 경조사비
• 점심 식대
• 교육비 지원

→ 상세 내용은 별도 안내
```

### Q5. 야근이 많나요?

```
A: 원칙적으로 없습니다! 🌙

원칙:
• 정시 퇴근이 기본
• 야근 필요 시 사전 협의
• 야근 시 대휴 or 수당

예외:
• 교회 행사 (주말)
• 컨퍼런스 (연 1-2회)
• 긴급 상황 (드물음)

→ 일과 삶의 균형 최우선
```

### Q6. 교회 방문이 부담스러워요

```
A: 처음엔 누구나 그렇습니다! 😊

준비:
• 교회 방문 에티켓 교육
• 시니어와 동행 (처음 3회)
• 사전 리허설
• 스크립트 제공

팁:
• 경청이 80%
• 목사님 존중
• 솔직하게 "배우는 중입니다"
• 완벽하지 않아도 OK

→ 경험 쌓으면 자연스러워짐
```

### Q7. 신앙이 필수인가요?

```
A: 존중하지만 강요하지 않습니다! 🙏

원칙:
• 신앙 유무와 업무 능력은 별개
• 다양한 배경 환영
• 종교 활동 강요 절대 없음

단,
• Mission (하나님 나라 확장) 공감 필요
• 교회 존중 필수
• 성경적 가치관 이해 필요

→ "같은 방향을 보는가"가 핵심
```
