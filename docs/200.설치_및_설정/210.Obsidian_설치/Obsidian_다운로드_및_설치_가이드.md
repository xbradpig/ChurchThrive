## 1. Obsidian 소개

### 🤔 Obsidian이란?

**Obsidian**은 마크다운(Markdown) 기반의 **지식 관리 도구(PKM - Personal Knowledge Management)**입니다.

**핵심 특징**:
- ✅ **완전 무료** (개인 사용)
- ✅ **오프라인 작동** (인터넷 없어도 됨)
- ✅ **내 컴퓨터에 저장** (클라우드 강제 아님)
- ✅ **평문 파일** (.md) - 영구 보존
- ✅ **강력한 링크 시스템** - 노트 간 연결
- ✅ **수천 개 플러그인** - 무한 확장

### 🏛️ 교회에서 왜 Obsidian을 쓰나요?

**전통적 방식의 문제**:
```
❌ 한글: 버전 관리 어려움, 검색 느림
❌ 워드: 파일이 커지면 느려짐
❌ 노션: 인터넷 필수, 데이터 소유권 불확실
❌ 에버노트: 비쌈, 락인(Lock-in)
```

**Obsidian의 장점**:
```
✅ 빠른 검색: 1,000개 문서도 0.1초
✅ 링크로 연결: 주보 ↔ 설교 ↔ 회의록
✅ AI 통합: ChatGPT, Claude와 완벽 호환
✅ 평생 무료: 개인 사용 영구 무료
✅ 데이터 소유: 내 컴퓨터에 100% 저장
```

### 💰 가격 정책

| 용도 | 가격 | 설명 |
|------|------|------|
| **개인 사용** | **무료** | 교회 업무도 개인 사용 ✅ |
| 상업 사용 | $50/년 | 회사 업무 (2인 이상) |
| Sync (공식 동기화) | $10/월 | OneDrive 쓰면 불필요 |
| Publish | $20/월 | 웹 발행 (거의 안 씀) |

**교회는 "개인 사용"으로 간주 → 완전 무료!** 🎉

## 3. Windows 설치 가이드

### Step 1: 다운로드 (3분)

1. **웹 브라우저 실행**
   - Chrome, Edge, Firefox 등 아무거나

2. **공식 사이트 접속**
   ```
   https://obsidian.md
   ```
   - 주소 정확히 입력 (obsidian.md)
   - 가짜 사이트 주의!

3. **Download 버튼 클릭**
   - 메인 페이지 중앙의 큰 버튼
   - 보라색 "Download" 버튼

4. **Windows 선택**
   - "Get it for Windows" 클릭
   - 또는 "Download for Windows" 클릭
   - 자동으로 `.exe` 파일 다운로드 시작

5. **다운로드 파일 확인**
   ```
   파일명: Obsidian-1.5.3.exe (버전은 달라질 수 있음)
   크기: 약 80-100MB
   위치: 다운로드 폴더
   ```

### Step 2: 설치 (5분)

1. **다운로드 폴더 열기**
   - `Windows키 + E` (파일 탐색기)
   - 왼쪽에서 "다운로드" 클릭

2. **설치 파일 실행**
   - `Obsidian-X.X.X.exe` 더블클릭
   - 또는 우클릭 → "관리자 권한으로 실행"

3. **보안 경고 처리**
   - "Windows의 PC 보호" 창이 뜨면:
   - "추가 정보" 클릭
   - "실행" 버튼 클릭

4. **설치 진행**
   - 자동으로 설치 시작
   - 설치 위치: `C:\Users\[사용자명]\AppData\Local\Obsidian\`
   - 시간: 약 1-2분

5. **설치 완료**
   - "Obsidian을 시작합니다" 체크
   - "완료" 버튼 클릭
   - Obsidian 자동 실행

### Step 3: 첫 실행 확인

**성공 화면**:
```
┌─────────────────────────────────┐
│  Obsidian                        │
├─────────────────────────────────┤
│                                  │
│  Open folder as vault            │
│  Create new vault                │
│  Open vault from Obsidian Sync   │
│                                  │
└─────────────────────────────────┘
```

**축하합니다! 설치 완료!** 🎉

## 4. Mac 설치 가이드

### Step 1: 다운로드 (3분)

1. **Safari 또는 Chrome 실행**

2. **공식 사이트 접속**
   ```
   https://obsidian.md
   ```

3. **Download 버튼 클릭**

4. **macOS 선택**
   - "Get it for macOS" 클릭
   - **Intel Mac**: "Intel" 버전
   - **M1/M2/M3 Mac**: "Apple Silicon" 버전
   - *잘 모르면: "Universal" 버전 (둘 다 작동)*

5. **다운로드 확인**
   ```
   파일명: Obsidian-1.5.3.dmg
   크기: 약 100-120MB
   위치: 다운로드 폴더
   ```

### Step 2: 설치 (5분)

1. **다운로드 폴더에서 .dmg 파일 열기**
   - `Obsidian-X.X.X.dmg` 더블클릭

2. **설치 창 나타남**
   ```
   ┌─────────────────────┐
   │  Obsidian 아이콘    │
   │         ↓           │
   │  Applications 폴더  │
   └─────────────────────┘
   ```

3. **Obsidian 아이콘을 Applications 폴더로 드래그**
   - 마우스로 끌어다 놓기
   - 복사 진행 (약 10초)

4. **설치 완료**
   - `.dmg` 파일 우클릭 → "꺼내기"
   - 또는 그냥 닫기

### Step 3: 첫 실행 (보안 처리)

1. **Launchpad 또는 Applications 폴더에서 Obsidian 클릭**

2. **보안 경고 나타남**
   ```
   "Obsidian은(는) 확인되지 않은 개발자가
   배포했기 때문에 열 수 없습니다."
   ```

3. **해결 방법 A (권장)**:
   - Obsidian 아이콘 **우클릭** (또는 Ctrl+클릭)
   - **"열기"** 선택
   - "열기" 버튼 다시 클릭
   - 이후부터는 정상 실행

4. **해결 방법 B (대안)**:
   - 시스템 환경설정 → 보안 및 개인 정보 보호
   - 하단에 "확인 없이 열기" 버튼 클릭

5. **첫 실행 성공!**

## 7. 첫 실행 및 Vault 생성

### 🗂️ Vault란?

> **Vault = 모든 노트가 저장되는 폴더**

**비유**:
- Vault = 도서관 건물
- 노트 = 책들
- 폴더 = 서가

**중요**:
- Vault는 여러 개 만들 수 있음
- 각 Vault는 독립적
- 교회 용도라면 Vault 1개면 충분

---

### 방법 1: 새 Vault 생성 (권장)

**Windows/Mac 공통**:

1. **"Create new vault" 클릭**

2. **설정 입력**:
   ```
   Vault name: 우리교회
   (또는 실제 교회 이름: 새소망교회, 은혜교회 등)

   Location:
   - Windows: C:\Users\[사용자명]\Documents\
   - Mac: /Users/[사용자명]/Documents/

   (기본값 그대로 두면 됨)
   ```

3. **"Create" 버튼 클릭**

4. **Vault 생성 완료!**
   - 빈 화면 나타남
   - 왼쪽: 파일 목록 (비어있음)
   - 오른쪽: 편집 영역

---

### 방법 2: 클라우드 폴더를 Vault로 사용 (팀 협업용)

**OneDrive 또는 Google Drive에 Vault 생성**

#### OneDrive 사용 시

1. **OneDrive 폴더 확인**
   - Windows: `C:\Users\[사용자명]\OneDrive\`
   - Mac: `/Users/[사용자명]/OneDrive/`

2. **OneDrive에 폴더 생성**
   - `OneDrive/우리교회` 폴더 만들기
   - 파일 탐색기에서 직접 생성

3. **Obsidian에서 폴더 열기**
   - "Open folder as vault" 클릭
   - `OneDrive/우리교회` 선택
   - "Open" 클릭

4. **완료!**
   - 이제 OneDrive가 자동 동기화
   - 다른 컴퓨터에서도 동일한 폴더 열면 됨

**장점**:
- ✅ 자동 백업
- ✅ 팀원들과 실시간 공유
- ✅ 여러 기기에서 동기화

---

### 첫 노트 만들기 (1분)

1. **왼쪽 상단 "New note" 아이콘 클릭**
   - 또는 `Ctrl+N` (Windows)
   - 또는 `Cmd+N` (Mac)

2. **제목 입력**:
   ```
   환영합니다
   ```

3. **내용 입력**:
   ```markdown
   # 우리 교회 Obsidian Vault에 오신 것을 환영합니다!

   오늘부터 모든 교회 문서를 여기에서 관리합니다.

   ## 주요 카테고리
   - [[주보]]
   - [[설교]]
   - [[회의록]]

   ## 다음 할 일
   - [ ] 폴더 구조 만들기
   - [ ] 주보 템플릿 생성
   - [ ] 첫 주보 작성
   ```

4. **저장**
   - `Ctrl+S` (Windows) 또는 `Cmd+S` (Mac)
   - Obsidian은 자동 저장되므로 생략 가능

**축하합니다! 첫 노트 완성!** 🎉

## 8. 문제 해결

### 문제 1: Windows에서 "앱을 시작할 수 없습니다" 오류

**증상**:
```
"이 앱을 시작할 수 없습니다.
0xc0000142 오류"
```

**해결**:

1. **Visual C++ Redistributable 설치**
   - https://aka.ms/vs/17/release/vc_redist.x64.exe
   - 다운로드 및 설치

2. **Windows 업데이트**
   - 설정 → 업데이트 및 보안
   - "업데이트 확인"

3. **재부팅**

---

### 문제 2: Mac에서 "손상되었기 때문에 열 수 없습니다" 오류

**증상**:
```
"Obsidian은(는) 손상되었기 때문에
열 수 없습니다."
```

**해결 (터미널 명령어)**:

1. **터미널 열기**
   - Spotlight (`Cmd+Space`)
   - "터미널" 입력 및 실행

2. **명령어 입력**:
   ```bash
   sudo xattr -cr /Applications/Obsidian.app
   ```

3. **비밀번호 입력**
   - Mac 관리자 비밀번호 입력 (화면에 안 보임)
   - Enter

4. **Obsidian 다시 실행**

---

### 문제 3: 한글이 깨져요 (��� 로 보임)

**해결**:

1. **Settings 열기**
   - 톱니바퀴 아이콘 클릭
   - 또는 `Ctrl+,` (Windows) / `Cmd+,` (Mac)

2. **Editor 메뉴**
   - 왼쪽에서 "Editor" 클릭

3. **Font 설정**
   - "Font family" 찾기
   - 입력: `Nanum Gothic, 맑은 고딕, 나눔고딕`

4. **적용 확인**

---

### 문제 4: OneDrive 동기화가 안 돼요

**확인 사항**:

1. **OneDrive 실행 중인가?**
   - 작업 표시줄에 OneDrive 아이콘 확인
   - 클릭 → "동기화 중" 또는 "최신 상태"

2. **파일이 OneDrive 폴더에 있나?**
   - Vault 위치 확인
   - `C:\Users\[사용자명]\OneDrive\우리교회\`

3. **OneDrive 용량 충분한가?**
   - OneDrive 설정 → 저장 공간 확인
   - 무료: 5GB, 유료: 1TB

**해결**:
- OneDrive 재시작
- 컴퓨터 재부팅
- OneDrive 다시 로그인

### 문제 6: 설치는 됐는데 실행이 너무 느려요

**원인**:
- 오래된 컴퓨터
- RAM 부족
- 다른 프로그램 많이 실행 중

**해결**:

1. **Community plugins 비활성화**
   - Settings → Community plugins
   - "Safe mode" 켜기

2. **불필요한 프로그램 종료**
   - `Ctrl+Shift+Esc` (작업 관리자)
   - 메모리 많이 쓰는 프로그램 종료

3. **Vault 크기 줄이기**
   - 큰 이미지 파일 삭제
   - 오래된 문서 아카이브

## ❓ 자주 묻는 질문 (FAQ)

### Q1: Obsidian은 정말 무료인가요?

**A**: 네! 개인 사용은 영구 무료입니다.
- 교회 사용 = 개인 사용 ✅
- 무제한 노트, 무제한 Vault
- 모든 기능 사용 가능

### Q2: 인터넷 없어도 되나요?

**A**: 네! 완전히 오프라인에서 작동합니다.
- 설치 시에만 인터넷 필요
- 사용 시에는 인터넷 불필요
- 클라우드 동기화도 선택사항

### Q3: 데이터가 어디에 저장되나요?

**A**: 100% 내 컴퓨터에 저장됩니다.
- Obsidian 회사가 데이터를 보지 않음
- 외부 서버로 전송 안 됨
- 완전한 프라이버시

### Q4: 나중에 다른 프로그램으로 옮길 수 있나요?

**A**: 네! 평문 마크다운 파일이라 언제든 이동 가능합니다.
- 노션, 에버노트로 import 가능
- 한글, 워드로 변환 가능
- Lock-in 없음

### Q5: 스마트폰과 PC를 동시에 쓸 수 있나요?

**A**: 네! OneDrive나 Google Drive로 동기화하면 됩니다.
- PC에서 작성 → 자동 동기화
- 스마트폰에서 즉시 확인 가능
- 실시간 협업 가능

### Q6: 팀원들과 공유하려면 어떻게 하나요?

**A**: 3가지 방법이 있습니다:
1. **OneDrive 공유 폴더** (권장, 무료)
2. **Google Drive 공유 폴더** (무료)
3. **Obsidian Sync** (공식, $10/월)

### Q7: 업데이트는 어떻게 하나요?

**A**: 자동 업데이트됩니다.
- 새 버전 나오면 알림
- 클릭 한 번으로 업데이트
- 또는 수동 다운로드

### Q8: 삭제하려면 어떻게 하나요?

**A**: 일반 프로그램 삭제와 동일합니다.
- **Windows**: 제어판 → 프로그램 제거
- **Mac**: Applications 폴더에서 휴지통으로 드래그
- Vault 폴더는 별도로 삭제 필요 (문서 보존 원하면 그대로 두기)

## 💡 Pro Tips

### Tip 1: 여러 Vault 만들기

```
용도별 Vault 분리:
- Vault 1: 교회 업무 (주보, 설교, 회의록)
- Vault 2: 개인 노트 (성경 묵상, 기도 제목)
- Vault 3: 프로젝트 (건축, 특별 사역)
```

**장점**: 깔끔한 관리, 빠른 검색

### Tip 2: 바탕화면 단축 아이콘 만들기

**Windows**:
1. 바탕화면 우클릭 → 새로 만들기 → 바로 가기
2. 위치: `C:\Users\[사용자명]\AppData\Local\Obsidian\Obsidian.exe`

**Mac**:
1. Applications 폴더에서 Obsidian 찾기
2. Dock로 드래그

### Tip 3: 여러 창 띄우기

- `Ctrl+Shift+N` (Windows)
- `Cmd+Shift+N` (Mac)

**용도**: 주보 보면서 설교 노트 작성
