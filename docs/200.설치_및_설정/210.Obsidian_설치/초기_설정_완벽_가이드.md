## 1. 설정 화면 열기

### 방법 1: 아이콘 클릭
- 왼쪽 하단 **톱니바퀴 ⚙️** 아이콘 클릭

### 방법 2: 단축키
- **Windows**: `Ctrl + ,`
- **Mac**: `Cmd + ,`

### 설정 화면 구조
```
┌────────────────┬─────────────────────────┐
│ Settings       │                         │
│                │                         │

│ Files & Links  │                         │
│ Appearance     │                         │
│ Hotkeys        │                         │
│ Core plugins   │                         │
│ ...            │                         │
└────────────────┴─────────────────────────┘
```

## 2. 필수 설정 (반드시 해야 함)

### 2.1 파일 & 링크 설정

**Settings → Files & Links**

#### ✅ 새 파일 위치 설정
```
Default location for new notes:
→ "In the folder specified below" 선택

Folder to create new notes in:
→ "100.교회운영/110.주보" 입력

(또는 가장 자주 만드는 폴더)
```

**이유**: 새 노트가 항상 원하는 폴더에 생성됨

#### ✅ 링크 설정
```
New link format:
→ "Shortest path when possible" 선택
```

**이유**: 링크가 깔끔해짐
- Before: `[[100.교회운영/110.주보/2025년 11월 17일 주보]]`
- After: `[[2025년 11월 17일 주보]]`

#### ✅ 첨부 파일 위치
```
Default location for new attachments:
→ "In subfolder under current folder" 선택

Subfolder name:
→ "attachments" 입력
```

**이유**: 이미지 파일이 깔끔하게 정리됨
```
110.주보/
├── 2025년 11월 17일 주보.md
└── attachments/
    ├── 교회_전경.jpg
    └── 행사_포스터.png
```

---

### 2.2 편집기 설정

**Settings → Editor**

#### ✅ 자동 저장
```
Auto pair brackets:
→ ✅ 켜기 (체크)

Auto pair Markdown syntax:
→ ✅ 켜기 (체크)
```

**효과**: `[[` 입력하면 자동으로 `]]` 완성

#### ✅ 줄 번호 표시 (선택)
```
Show line number:
→ ✅ 켜기 (선택사항)
```

**효과**: 긴 문서 편집 시 유용

#### ✅ 읽기 모드 자동 전환 (선택)
```
Default view for new tabs:
→ "Reading view" 또는 "Editing view" 선택

추천: "Editing view" (편집 우선)
```

---

### 2.3 외관 설정

**Settings → Appearance**

#### ✅ 언어 설정
```
Language:
→ "한국어" 선택
→ Obsidian 재시작
```

**효과**: 모든 메뉴가 한글로 표시

#### ✅ 테마 선택
```
Base color scheme:
→ "Adapt to system" (시스템 따라감)
→ "Light" (밝은 테마)
→ "Dark" (어두운 테마)

추천: "Light" (교회에서 프로젝터 사용 시 잘 보임)
```

#### ✅ 글꼴 설정
```
Font:
→ "나눔고딕, Nanum Gothic, 맑은 고딕" 입력

Font size:
→ "16" (기본값)
→ 목회자 연세 고려하여 18-20 추천
```

**중요**: 한글 폰트를 반드시 설정해야 깨지지 않음!

## 3. 추천 설정 (강력 권장)

### 3.1 Core Plugins 활성화

**Settings → Core plugins**

**반드시 켜야 할 플러그인 5개**:

#### ✅ 1. File Explorer
```
→ 이미 켜져 있음 (왼쪽 파일 목록)
```

#### ✅ 2. Search
```
→ 이미 켜져 있음 (검색 기능)
```

#### ✅ 3. Quick Switcher
```
→ ✅ 켜기

단축키: Ctrl+O (Windows) / Cmd+O (Mac)
```

**효과**: 파일 이름만 입력하면 즉시 열기

#### ✅ 4. Graph View
```
→ ✅ 켜기
```

**효과**: 노트 간 연결 관계를 그래프로 시각화

#### ✅ 5. Templates
```
→ ✅ 켜기
```

**효과**: 템플릿 사용 가능 (주보, 설교 노트 등)

**Templates 설정**:
```
Settings → Core plugins → Templates → 옵션 버튼

Template folder location:
→ "Templates" 입력
```

이제 `Templates/` 폴더에 템플릿 저장하면 됨

---

### 3.2 Daily Notes 설정 (선택)

**Settings → Core plugins → Daily notes**

```
→ ✅ 켜기 (선택사항)

Date format:
→ "YYYY-MM-DD" (ISO 표준)

New file location:
→ "Daily/" (일일 노트 전용 폴더)

Template file location:
→ "Templates/Daily Template"
```

**용도**: 매일 업무 일지, 기도 제목 등

## 4. 선택 설정 (취향에 맞게)

### 4.1 백링크 표시

**Settings → Core plugins → Backlinks**

```
→ ✅ 켜기

Backlink in document:
→ ✅ 켜기
```

**효과**: 현재 문서를 참조하는 다른 문서 자동 표시

**예시**:
```
[[2025년 11월 17일 주보]] 문서에서

백링크 (이 문서를 참조하는 문서):
- [[2025년 11월 회의록]]
- [[11월 출석 통계]]
```

---

### 4.2 창 분할 설정

**Settings → Core plugins → Window splits**

```
→ ✅ 켜기

Use tabs:
→ ✅ 켜기
```

**효과**: 여러 노트를 탭으로 전환하며 작업 가능

**사용법**:
- 주보 보면서 설교 노트 참조
- Ctrl+클릭으로 새 탭에서 열기

---

### 4.3 자동 백업 (File Recovery)

**Settings → Core plugins → File recovery**

```
→ ✅ 켜기 (이미 켜져 있음)

Snapshot interval:
→ "5" (5분마다 자동 저장)

History length:
→ "7" (7일치 보관)
```

**효과**: 실수로 삭제하거나 잘못 수정해도 복구 가능

## 5. 교회 특화 설정

### 5.1 폴더 구조 생성

**파일 탐색기에서 폴더 생성**:

```
우리교회/
├── 100.교회운영/
│   ├── 110.주보/
│   ├── 120.설교/
│   ├── 130.회의록/
│   ├── 140.행사/
│   ├── 150.교육/
│   ├── 160.심방/
│   ├── 170.새가족/
│   ├── 180.재정/
│   └── 190.자료실/
├── Templates/
│   ├── 주보_템플릿.md
│   ├── 설교_노트_템플릿.md
│   └── 회의록_템플릿.md
└── Daily/
```

**생성 방법**:
1. 파일 탐색기에서 우클릭
2. "New folder" 선택
3. 폴더 이름 입력
4. 반복

**Templates 폴더 생성 후**:

1. `Templates` 폴더 만들기

## 6. 단축키 설정

### 6.1 필수 단축키 익히기

**Settings → Hotkeys**

**기본 제공 단축키** (변경 불필요):

| 기능 | Windows | Mac | 설명 |
|------|---------|-----|------|
| **새 노트** | `Ctrl+N` | `Cmd+N` | 새 문서 생성 |
| **검색** | `Ctrl+Shift+F` | `Cmd+Shift+F` | 전체 검색 |
| **Quick Switcher** | `Ctrl+O` | `Cmd+O` | 파일 빠르게 열기 |
| **명령 팔레트** | `Ctrl+P` | `Cmd+P` | 모든 명령 검색 |
| **설정** | `Ctrl+,` | `Cmd+,` | 설정 열기 |
| **읽기/편집 전환** | `Ctrl+E` | `Cmd+E` | 모드 전환 |
| **창 분할** | `Ctrl+\` | `Cmd+\` | 좌우 분할 |

---

### 6.2 커스텀 단축키 추가 (선택)

**Settings → Hotkeys → 검색창에 입력**

#### 추천 커스텀 단축키:

**1. Insert template**
```
검색: "Insert template"
단축키 설정: Ctrl+T (Windows) / Cmd+T (Mac)
```

**2. Toggle fold**
```
검색: "Fold all headings"
단축키: Ctrl+Shift+[ (접기)

검색: "Unfold all headings"
단축키: Ctrl+Shift+] (펴기)
```

**3. Format bold**
```
검색: "Toggle bold"
단축키: Ctrl+B (이미 설정됨)
```

## ❓ 자주 묻는 질문

### Q1: 설정이 저장 안 돼요

**A**: 설정은 자동 저장됩니다.
- `.obsidian/` 폴더에 자동 저장
- Vault 폴더를 삭제하지 않는 한 유지됨

### Q2: 다른 컴퓨터에서 같은 설정 쓰고 싶어요

**A**: `.obsidian/` 폴더를 복사하세요.
1. Vault 폴더를 OneDrive에 두면 자동 동기화
2. 또는 `.obsidian/` 폴더만 USB로 복사

### Q3: 설정을 초기화하고 싶어요

**A**: `.obsidian/` 폴더를 삭제하면 됩니다.
- Vault 폴더에서 `.obsidian` 폴더 삭제
- Obsidian 재실행하면 기본 설정으로 시작

### Q4: 플러그인이 너무 많아서 헷갈려요

**A**: 처음엔 Core plugins만 사용하세요.
- Community plugins는 나중에 추가
- 필요할 때만 하나씩 켜기

### Q5: 폰트가 계속 깨져요

**A**: 시스템 폰트 확인하세요.
- Windows: 나눔고딕 설치 필요
- Mac: 기본 설치됨
- 또는 "맑은 고딕" 사용
