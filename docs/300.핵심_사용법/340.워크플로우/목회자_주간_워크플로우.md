## 주간 워크플로우 개요

### 📅 일주일 전체 흐름

```
월요일 (휴식일):
  - 지난 주 회고
  - 이번 주 계획
  - 설교 본문 선택

화요일 (기획일):
  - 설교 준비 시작
  - 심방 계획
  - 회의 준비

수요일 (집중일):
  - 설교 작성 집중
  - 수요예배 준비

목요일 (사역일):
  - 심방 및 상담
  - 설교 완성
  - 주보 초안

금요일 (마무리일):
  - 주보 최종 확인
  - 주일 준비 점검
  - 한 주 정리

토요일 (점검일):
  - 설교 최종 점검
  - 예배 리허설
  - 기도

주일 (예배일):
  - 예배 인도
  - 예배 후 기록
  - 피드백 수집
```

#### 1. 지난 주일 회고 (Obsidian)

**Step 1: 주일 예배 기록 열기**

파일: `100.교회운영/120.설교/2025-01-05 하나님의 사랑.md`

```markdown
## 예배 후 회고

### 잘된 점 ✅
- 성도들 반응 좋음
- 예화가 적절했음
- 시간 안배 완벽 (정확히 40분)

### 아쉬운 점 ⚠️
- 마무리가 다소 급했음
- 음향 문제 (마이크 터짐)

### 다음 주 개선사항
- 마지막 5분 여유 두기
- 음향 사전 점검 강화
```

## Obsidian 활용 팁

### 1. Daily Note 플러그인 활용

**설정**:
- Settings → Daily notes
- Template: `Templates/Daily Note Template.md`
- Format: `YYYY-MM-DD`

**자동 생성**:
- 매일 아침 Obsidian 열면 자동으로 오늘 노트 생성
- `Ctrl+D` 또는 `Cmd+D`

---

### 2. Calendar 플러그인으로 주간 보기

**사용법**:
1. 좌측 캘린더 아이콘 클릭
2. 이번 주 한눈에 보기
3. 날짜 클릭 → 해당 일 노트로 이동

---

### 3. Templater로 자동화

**주간 계획 템플릿**:

````markdown
---
type: weekly_plan
date: <% tp.date.now("YYYY-MM-DD") %>
week_number: <% tp.date.now("ww") %>
---

# <% tp.date.now("YYYY년 W주차 계획", "ww") %>

## 이번 주 핵심 목표

1. **설교**:
2. **심방**:
3. **행사**:

## 일정

<% tp.file.cursor() %>
````

---

### 4. Dataview로 한 주 요약

**내 할 일 보기**:

```dataview
TASK
FROM ""
WHERE !completed
AND due <= date(today) + dur(7 days)
SORT due ASC
```

**이번 주 심방 목록**:

```dataview
TABLE visitee, visit_type
FROM "200.성도관리/220.심방기록"
WHERE date >= date(today) - dur(7 days)
  AND date <= date(today)
SORT date DESC
```
