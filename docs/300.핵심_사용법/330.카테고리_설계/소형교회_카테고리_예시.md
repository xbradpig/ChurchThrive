## 소형교회 특징

### 교회 프로필

```yaml
교회명: 새소망교회
출석 인원: 70명
담임목사: 1명
부목사: 0명
사역자: 0명
장로: 2명
안수집사: 3명
권사: 5명

부서:
  - 주일학교 (10명)
  - 청년부 (15명)
  - 장년부 (45명)

주간 정기 모임:
  - 주일 예배 (2부)
  - 수요 기도회
  - 청년부 모임 (토요일)
```

### 문서 관리 현황

**주간 생성 문서**:
- 주보: 1개
- 설교 노트: 2개 (주일, 수요)

**월간 생성 문서**:
- 회의록: 1-2개 (운영위원회, 월 1회)
- 재정 보고: 1개

**연간 생성 문서**:
- 총회 자료: 1개
- 행사 기획서: 5-10개

**연간 총 문서**: 약 **150-200개**

```

**왜 주보+설교를 합쳤나?**

- 주보와 설교 밀접한 관계
- 폴더 단순화

**성장 후 분리**:
```

## 복사해서 바로 쓰는 구조

### 터미널에서 한 번에 생성 (Mac/Linux)

```bash
cd /path/to/vault

# 전체 폴더 구조 생성
mkdir -p 100.교회운영/{110.주보_설교,120.회의록,130.재정}
mkdir -p 200.성도관리/{210.등록_입교,220.심방기록}
mkdir -p 300.부서활동/{310.주일학교,320.청년부,330.장년부}
mkdir -p 400.행사/{410.정기행사,420.특별행사}
mkdir -p 900.개인노트/{910.묵상노트,920.설교준비}

echo "✅ 소형교회 폴더 구조 생성 완료!"
```

### Windows PowerShell

```powershell
cd C:\path\to\vault

# 폴더 생성
New-Item -ItemType Directory -Path "100.교회운영\110.주보_설교" -Force
New-Item -ItemType Directory -Path "100.교회운영\120.회의록" -Force
New-Item -ItemType Directory -Path "100.교회운영\130.재정" -Force
New-Item -ItemType Directory -Path "200.성도관리\210.등록_입교" -Force
New-Item -ItemType Directory -Path "200.성도관리\220.심방기록" -Force
New-Item -ItemType Directory -Path "300.부서활동\310.주일학교" -Force
New-Item -ItemType Directory -Path "300.부서활동\320.청년부" -Force
New-Item -ItemType Directory -Path "300.부서활동\330.장년부" -Force
New-Item -ItemType Directory -Path "400.행사\410.정기행사" -Force
New-Item -ItemType Directory -Path "400.행사\420.특별행사" -Force
New-Item -ItemType Directory -Path "900.개인노트\910.묵상노트" -Force
New-Item -ItemType Directory -Path "900.개인노트\920.설교준비" -Force

Write-Host "✅ 소형교회 폴더 구조 생성 완료!"
```

### Obsidian에서 수동 생성 (10분)

1. 좌측 파일 목록에서 **우클릭**
2. **"New folder"** 클릭
3. 폴더명 입력: `100.교회운영`
4. Enter
5. `100.교회운영` 우클릭 → "New folder" → `110.주보_설교`
6. 반복...
