# 중형교회 카테고리 구조 예시

> **실제 사례: 은혜중앙교회 (출석 200명)**
>
> 창립 10년차, 담임목사 + 부목사 1명, 사역자 5명, 부서 8개

---

## 교회 프로필

```yaml
교회명: 은혜중앙교회
출석 인원: 200명
담임목사: 1명
부목사: 1명 (교육 담당)
사역자: 5명 (찬양, 미디어, 청년, 행정, 재정)

부서:
  - 유년부 (20명)
  - 초등부 (25명)
  - 중고등부 (30명)
  - 청년부 (40명)
  - 장년부 (85명)
  - 찬양팀 (15명)
  - 미디어팀 (5명)
  - 구역모임 (5개 구역)

연간 문서: 약 800-1000개
```

## 주요 특징

### 1. 설교 분리 (목회자별)

```
120.설교/
  담임목사/
    2025-01-05 하나님의 사랑.md
    2025-01-12 믿음의 여정.md
  부목사/
    2025-01-08 수요예배_기도.md
    2025-01-15 수요예배_찬양.md
```

**이유**:
- 설교자 2명
- 검색 편의성
- 통계 분리

---

### 2. 교육 부서 세분화

```
300.교육사역/
  310.유년부/      (미취학)
  320.초등부/      (1-6학년)
  330.중고등부/    (중1-고3)
  340.청년부/      (20-30대)
  350.장년부/      (40대+)
  360.새가족반/
```

**각 부서별 문서**:
- 주간 보고
- 교육 계획
- 행사 기획

---

### 3. 구역 관리

```
230.구역/
  1구역/
    2025-01 1구역 모임.md
    1구역 명단.md
  2구역/
    ...
```

**활용**:
- 구역별 모임 기록
- 구역 출석 통계
- 구역장 보고

---

### 4. 연도별 아카이브

**대량 문서 폴더는 연도별 분리**:

```
110.주보/
  2025년/        ← 올해 (52개)
  2024년/        ← 작년 (52개)
  2023년/        ← 2년 전 (52개)
```

**소량 문서 폴더는 연도 분리 안 함**:

```
130.회의록/
  2025-01-14 운영위원회.md
  2025-02-14 운영위원회.md
  2024-12-14 운영위원회.md
```

## 연간 문서 예상량

| 카테고리 | 주간 | 연간 |
|----------|------|------|
| 주보 | 1 | ~52 |
| 설교 | 4 | ~200 |
| 회의록 | 1 | ~50 |
| 재정 | 0.25 | 15 |
| 심방 | 5 | ~250 |
| 교육 부서 | 6 | ~300 |
| 행사 | 0.5 | ~20 |
| 기타 | 2 | ~100 |
| **합계** | ~20 | **~987개** |

**저장 용량**: 연간 ~10MB

## 팀 협업 설정

### 폴더별 담당자

| 폴더 | 담당자 | 권한 |
|------|--------|------|
| 110.주보 | 행정사역자 | 편집 |
| 120.설교/담임목사 | 담임목사 | 편집 |
| 120.설교/부목사 | 부목사 | 편집 |
| 300.교육사역 | 부목사 | 총괄 |
| 400.선교전도 | 선교위원장 | 편집 |
| 510.찬양팀 | 찬양사역자 | 편집 |

### OneDrive 동기화 (추천)

**설정**:
1. Vault를 OneDrive 폴더에 생성
2. 모든 팀원 OneDrive 동기화
3. 동시 편집 금지 (시간대 분리)

**장점**:
- 자동 백업
- 팀원 간 실시간 공유
- 버전 관리
