# ChurchThrive 폴더 구조 최종 설계 (1-2-3 레이어)

## 📐 계층 규칙

### 1레이어 (대분류) - 100단위만
루트에만 존재: `000, 100, 200, 300, 400, 500, 600, 700, 800, 900`

### 2레이어 (중분류) - 1레이어 내부
각 1레이어 폴더 내부에 10단위로 존재
- `000/` 내부: `010/, 020/, 030/, 040/, ...`
- `100/` 내부: `110/, 120/, 130/, 140/, ...`
- `200/` 내부: `210/, 220/, 230/, 240/, ...`

### 3레이어 (소분류) - 2레이어 내부
각 2레이어 폴더 내부에 1단위로 존재
- `010/` 내부: `011/, 012/, 013/, ...`
- `110/` 내부: `111/, 112/, 113/, ...`
