# 참여자 사전 준비 가이드

> **"성공적인 부트캠프는 철저한 준비에서 시작됩니다"**
>
> Day 1 시작 전에 이 가이드의 모든 항목을 완료하세요.

---

## 📋 준비 체크리스트 (30개)

### ✅ 1단계: 하드웨어 준비 (5개)

```
 □ 컴퓨터 준비
   - Windows 10 이상
   - macOS 10.15 이상
   - Linux (Ubuntu, Fedora 등)

 □ 사양 확인
   - RAM 4GB 이상 (8GB 권장)
   - 저장공간 5GB 이상 여유
   - 인터넷 연결 가능

 □ 주변 기기
   - 마우스 (필수)
   - 듀얼 모니터 (권장)
   - 헤드셋 (선택)

 □ 백업 저장소
   - 외장 하드 또는
   - 클라우드 스토리지

 □ 전원 안정성
   - 노트북: 충전기
   - 데스크톱: 무정전전원장치 (권장)
```

### ✅ 2단계: 소프트웨어 설치 (8개)

#### 필수 설치

```
 □ Obsidian 다운로드
   https://obsidian.md/download
   ⏰ 5분 소요

 □ 웹 브라우저
   Chrome 또는 Edge 권장
   (Safari, Firefox도 가능)

 □ 압축 프로그램
   - Windows: 7-Zip
   - Mac: 기본 제공
```

#### 선택 설치 (필요 시)

```
 □ OneDrive
   팀 협업 시 필요
   https://www.microsoft.com/onedrive

 □ Google Drive Backup & Sync
   OneDrive 대신 사용 가능
   https://www.google.com/drive/download/

 □ GitHub Desktop
   웹 퍼블리싱 시 필요
   https://desktop.github.com/

 □ Visual Studio Code
   고급 마크다운 편집 시
   https://code.visualstudio.com/

 □ Typora
   외부 마크다운 에디터
   https://typora.io/
```

### ✅ 3단계: 계정 생성 (5개)

```
 □ Microsoft 계정
   - OneDrive 동기화용
   - 이미 있으면 SKIP
   - outlook.com 이메일

 □ Google 계정
   - Google Drive 동기화용
   - Gmail 이메일

 □ GitHub 계정 (선택)
   - 웹 퍼블리싱 시 필요
   - https://github.com/signup

 □ Notion 계정 (선택)
   - Notion 연동 시 필요
   - https://www.notion.so/signup

 □ ChurchThrive 커뮤니티
   - 질문 게시판 가입
   - Discord 또는 Slack
```

### ✅ 4단계: 자료 준비 (6개)

```
 □ 교회 과거 자료 수집
   - 주보 PDF/워드 10개
   - 회의록 5개
   - 설교노트 5개
   - 심방기록 3개

 □ 교회 기본 정보
   - 교회명
   - 주소, 연락처
   - 담임목사명
   - 설립 연도

 □ 교회 조직도
   - 당회/제직회 구성
   - 부서 리스트
   - 선교회 리스트

 □ 연간 일정
   - 주요 절기 (부활절, 성탄절 등)
   - 행사 일정
   - 수련회 일정

 □ 출석 통계 (있으면)
   - 최근 3개월 주일 출석
   - 구역별 출석

 □ 재정 정보 (있으면)
   - 월별 헌금 내역 (공개 가능 범위)
```

### ✅ 5단계: 환경 설정 (6개)

```
 □ 학습 공간 확보
   - 조용한 공간
   - 책상 + 의자
   - 적절한 조명

 □ 시간 블록 확보
   - 하루 3-4시간 고정
   - 15일 일정 확보
   - 가족/팀원에게 공지

 □ 방해 요소 제거
   - 스마트폰 알림 끄기
   - "학습 중" 안내문
   - 이메일 자동 응답 설정

 □ 학습 도구 준비
   - 노트 (종이 또는 디지털)
   - 펜/형광펜
   - 포스트잇

 □ 백업 계획
   - 일일 백업 시간 설정
   - 백업 저장소 확인

 □ 응급 연락망
   - 코치 연락처
   - 동료 학습자 연락처
   - 기술 지원 채널
```

## 🧠 학습 마인드셋

### ✅ 성공하는 참여자의 특징

```
1. 능동적
━━━━━━━━━━━━━━━━━━━━━━━━
• "어떻게 하면 우리 교회에 적용할까?"
• 가이드를 읽고 바로 실습
• 질문을 두려워하지 않음

2. 끈기
━━━━━━━━━━━━━━━━━━━━━━━━
• 어려워도 포기하지 않음
• 코치에게 도움 요청
• 동료와 함께 해결

3. 실용주의
━━━━━━━━━━━━━━━━━━━━━━━━
• 완벽보다 완성 추구
• 70% 이해하면 다음 단계
• 나중에 다시 돌아와 복습

4. 공유 정신
━━━━━━━━━━━━━━━━━━━━━━━━
• 배운 것을 동료와 공유
• 질문 게시판에서 답변
• 성공 경험을 나눔
```

### ❌ 실패하는 참여자의 특징

```
1. 수동적
━━━━━━━━━━━━━━━━━━━━━━━━
• "가이드만 읽으면 되겠지"
• 실습 과제를 건너뜀
• 질문하지 않음

2. 완벽주의
━━━━━━━━━━━━━━━━━━━━━━━━
• 100% 이해할 때까지 진도 안 나감
• 작은 오류에 집착
• 기한 내 완료 못함

3. 고립
━━━━━━━━━━━━━━━━━━━━━━━━
• 혼자 해결하려고만 함
• 커뮤니티 활용 안함
• 코치 피드백 무시

4. 일관성 부족
━━━━━━━━━━━━━━━━━━━━━━━━
• 오늘 4시간, 내일 0시간
• 주말에 몰아서 하려고 함
• 일정 관리 실패
```
