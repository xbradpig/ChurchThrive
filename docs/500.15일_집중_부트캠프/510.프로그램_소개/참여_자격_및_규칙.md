## 📚 선수 조건 (Prerequisites)

### 1. 기술적 준비

```yaml
최소_요구사항:

하드웨어:
  ✅ 노트북 or 데스크톱 (태블릿 불가)
  ✅ OS: Windows 10 이상 or macOS 10.15 이상
  ✅ RAM: 8GB 이상
  ✅ 저장공간: 10GB 이상 여유

소프트웨어:
  ✅ 인터넷 브라우저 (Chrome, Safari, Edge)
  ✅ Zoom (화상 회의)
  ✅ 압축 프로그램 (알집, 반디집 등)

네트워크:
  ✅ 안정적인 인터넷 (최소 10Mbps)
  ✅ WiFi or 유선 (학습 시간 중)

기본_능력:
  ✅ 파일 생성/저장/열기
  ✅ 폴더 만들기/이동
  ✅ 인터넷 검색
  ✅ 이메일 송수신
  ✅ 한글 타이핑 (분당 30타 이상)
```

### 2. 시간 확보

```yaml
필수_시간_투자:

일일_학습:
  하루: 3-4시간 (연속 또는 분할)
  총_15일: 45-60시간

시간_배분:
  이론_학습: 30% (1-1.5시간)
  실습: 50% (1.5-2시간)
  과제_정리: 20% (0.5-1시간)

권장_시간대:
  옵션1: 아침 05:00-08:00 (집중력 최고)
  옵션2: 낮 10:00-14:00 (업무 중)
  옵션3: 저녁 20:00-23:00 (대중적)

⚠️ 중요:
  - "틈틈이"는 안 됩니다
  - 고정 시간 확보 필수
  - 가족/직장 협조 사전 요청
```

### 3. 교회 데이터 준비

```yaml
사전_준비_자료:

필수:
  ☑️ 교인 명부 (이름, 연락처)
  ☑️ 최근 3개월 출석 기록
  ☑️ 교회 조직도 (있으면)
  ☑️ 연간 사역 계획 (있으면)

권장:
  ⭐ 주보 파일 (최근 10주)
  ⭐ 회의록 (최근 5회)
  ⭐ 과거 행사 자료
  ⭐ 재정 보고서 (최근 6개월)

형식:
  - 엑셀, 워드, PDF, 한글 모두 가능
  - 종이 문서는 스캔 or 사진 촬영

⚠️ 주의:
  - 개인정보 보호 필수
  - 코치에게 공유 시 동의 확인
```

### 4. 마음가짐

```yaml

필수:
```

```yaml
    - 질문 적극 권장
```

## 📊 출석 및 진도 관리

### 1. 출석 체크

```yaml
출석_방법:

일일_체크인:
  시간: 매일 00:00-23:59
  방법: 커뮤니티 "출석" 버튼
  내용:
    - 오늘 학습 시간
    - 완료한 내용
    - 어려운 점
    - 내일 계획

인증샷:
  - Obsidian 화면 스크린샷
  - 또는 학습 노트 사진
  - 커뮤니티 업로드

출석_인정:
  - 체크인 + 인증샷 → 출석
  - 체크인만 → 반출석
  - 아무것도 안 함 → 결석

결석_허용:
  - 15일 중 2일 이내 허용
  - 사전 통보 필요
  - 3일 이상 = 재수강 권장
```

### 2. 진도 관리

```yaml
진도_체크_포인트:

Day_1-3:
  체크: 기초 설정 완료 여부
  기준: Obsidian 설치, 첫 Vault 생성
```

