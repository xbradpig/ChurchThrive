## 1. 부트캠프 소개

### 🎯 미션

**"모든 교회가 체계적인 지식관리 시스템으로 건강하게 성장하도록 돕는다"**

### 🔥 부트캠프 철학

```
❌ 이론 중심 강의
❌ 수동적 학습
❌ 느슨한 진도 관리

✅ 실전 중심 실습 (70%)
✅ 능동적 학습 (매일 과제)
✅ 강력한 코칭 (일일 점검)
```

### 📊 예상 성과

**15일 후, 당신의 교회는:**

- ✅ **Obsidian 기반 지식관리 시스템** 완성
- ✅ **100+ 폴더 구조** 체계적으로 설계
- ✅ **회의록, 주보, 설교노트 템플릿** 7개 운영
- ✅ **주간/월간 워크플로우** 자동화
- ✅ **통계 대시보드** 실시간 운영
- ✅ **팀 협업 동기화** 설정 완료
- ✅ **과거 자료 20개 이상** 디지털화
- ✅ **ChurchThrive 공식 인증서** 발급

## 2. 왜 15일 집중 방식인가?

### 🚫 30일 분산 학습의 문제점

```
문제 1: 느슨한 진도
━━━━━━━━━━━━━━━━━━━━━━━━
하루 1-2시간 → 쉽게 미루게 됨
"내일 하지 뭐" → 완료율 ↓

문제 2: 긴 학습 주기
━━━━━━━━━━━━━━━━━━━━━━━━
30일 → 중간에 흥미 감소
복습 주기 ↑ → 학습 효율 ↓

문제 3: 낮은 몰입도
━━━━━━━━━━━━━━━━━━━━━━━━
하루 1시간 → 맥락 전환 비용 큼
집중력 발휘 어려움
```

### ✅ 15일 집중 부트캠프의 장점

```
장점 1: 높은 집중도
━━━━━━━━━━━━━━━━━━━━━━━━
하루 3-4시간 → 깊은 몰입
2주 집중 → 흥미 유지 가능

장점 2: 빠른 완성
━━━━━━━━━━━━━━━━━━━━━━━━
15일 → 빠른 성과 경험
성취감 ↑ → 완료율 ↑

장점 3: 강력한 코칭
━━━━━━━━━━━━━━━━━━━━━━━━
일일 점검 → 낙오 방지
중간 평가 → 진도 보장
```

### 📈 데이터 비교

| 구분 | 30일 분산 | 15일 집중 |
|-----|----------|----------|
| **일일 투자** | 1-2시간 | 3-4시간 |
| **총 투자 시간** | 30-60시간 | 45-60시간 |
| **완료율** | 60-70% | 80-90% |
| **집중도** | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| **코칭 빈도** | 주 1회 | 매일 |
| **중간 평가** | 없음 | 2회 (Day 7, 14) |
