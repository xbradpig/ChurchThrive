# 10. 개발팀 및 예산

## 개발팀 구성 (MVP)

| 역할 | 인원 | 기간 | 핵심 역량 |
|------|------|------|-----------|
| **테크리드/풀스택 시니어** | 1명 | 6개월+ | Supabase + Next.js 아키텍처, DB 설계, API 설계 |
| **프론트엔드 개발자** | 1명 | 6개월+ | React/Next.js, PWA, 반응형 웹, Tailwind CSS |
| **모바일 개발자** | 1명 | 6개월+ | React Native/Expo, iOS/Android 동시 개발 |
| **UI/UX 디자이너** | 1명 | 3개월 (계약) | 브랜드 가이드 기반 디자인 시스템, 접근성 |

### 역할 상세

**테크리드/풀스택 시니어**:
- Supabase 아키텍처 설계 (RLS, Edge Functions)
- DB 스키마 설계 및 마이그레이션
- STT 파이프라인 구축
- Google Cloud API 연동
- CI/CD 파이프라인 구축
- 코드 리뷰 및 기술 의사결정

**프론트엔드 개발자**:
- Next.js 14+ 웹 앱 개발
- PWA 구현 (Service Worker, 오프라인)
- 교회 홈페이지 빌더 (SSG)
- 반응형 UI (모바일~데스크톱)
- Supabase 클라이언트 연동

**모바일 개발자**:
- React Native (Expo) 앱 개발
- iOS/Android 동시 개발
- 오프라인 녹음 기능
- 푸시 알림 (FCM/APNs)
- 앱 스토어 배포

**UI/UX 디자이너**:
- ChurchThrive 브랜드 가이드 기반 디자인 시스템
- 와이어프레임 → 프로토타입 → 최종 UI
- 접근성 설계 (WCAG 2.1 AA)
- 고령 교인 UX 최적화

> **Supabase 활용으로 별도 백엔드 개발자 불필요**: Supabase가 DB, Auth, Storage, Realtime, Edge Functions를 제공하므로 백엔드 인력 절감.
