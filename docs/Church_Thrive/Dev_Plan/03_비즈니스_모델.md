# 03. 비즈니스 모델

## 수익 모델: Freemium SaaS

교회 규모에 따른 단계적 과금 + STT 종량제 + 컨설팅 번들

---

## 가격 전략 근거

### 무료 티어 (씨앗) 전략
- **목표**: 진입장벽 제거, 최대 교회 등록 유도
- 50명 이하 소규모 교회 = 완전 무료 사용 가능
- 말씀노트(텍스트) + 교인관리 + 교회홈페이지 = 무료로 충분한 가치 제공
- **전환 트리거**: STT 기능, 교인 수 50명 초과

### 유료 전환 심리
1. 무료로 교인관리 + 말씀노트 사용 → 습관 형성
2. STT 기능 체험 (체험판 or 첫 달 무료) → 가치 인지
3. 교인 수 증가 → 자연스러운 상위 티어 전환
4. 재정관리/전자결재 필요 → Standard 전환

### 경쟁사 대비 포지셔닝
- **디바인/지저스온**: 기능 비슷하지만 AI 없음, 모바일 약함
- **ChurchThrive**: 동일 가격대에서 AI STT + 영적성장 도구 + 홈페이지 **추가 제공**
- **차별 가치**: STT만으로 월 수만원 가치 (외부 STT 서비스 대비)

## 매출 성장 레버

1. **무료→유료 전환**: STT 체험 → 유료 전환 (핵심 드라이버)
2. **업셀**: Basic → Standard → Premium (교회 성장에 따른 자연 업그레이드)
3. **STT 종량**: 설교 횟수 증가 → STT 추가 구매
4. **컨설팅 번들**: 앱 + 부트캠프/LVI 통합 패키지
5. **교단 일괄 계약**: 교단 MOU로 대량 도입 (디스카운트 적용)
