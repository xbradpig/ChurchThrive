## 프로젝트 한줄 요약

교인관리, 교회 행정관리, 말씀노트(녹음→STT + 텍스트), 훈련관리, 사역관리를 통합한 **교회 총괄 관리 SaaS 앱**

**핵심 차별화**: AI 기반 한국어 설교 음성→텍스트(STT) 변환 + 기존 ChurchThrive 컨설팅 생태계 통합

## 개발 Phase 요약

| Phase | 기간 | 핵심 기능 |
|-------|------|-----------|
| **Phase 1 - MVP** | Month 1-4 | 가입/인증, 교인관리, 말씀노트, 행정관리(기본), 교회홈페이지 자동빌드, 구역/셀 관리 |
| **Phase 2 - Growth** | Month 5-7 | 한국어 STT, 전자결재, 재정관리, 훈련관리, 말씀 암송/통독, 구독결제 |
| **Phase 3 - Maturity** | Month 8-10 | LVI 교회건강진단, 봉사스케줄링, 회의록, 캘린더/SMS 연동 |
| **Phase 4 - Enterprise** | Month 11-16 | 멀티캠퍼스, AI 추천, 화이트라벨, API 공개, 영어 지원 |

---

## 기술 스택 요약

- **프론트엔드**: Next.js 14+ (PWA) + React Native (Expo)
- **호스팅**: Cloudflare Pages (무료)
- **백엔드**: Supabase (PostgreSQL + Auth + Storage + Realtime + Edge Functions)
- **STT**: Google Cloud STT (1순위) + Whisper API (2순위) + CLOVA (3순위)
- **인증**: Supabase Auth + Kakao OAuth
