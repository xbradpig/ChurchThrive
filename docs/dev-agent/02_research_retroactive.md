# 02. Research (리서치) - 역방향 문서화

## 요약

시장 분석, 사용자 리서치, GTM 전략 문서를 통해 리서치 스테이지가 완료되었습니다.
한국 개신교 교회 관리 SaaS 시장의 경쟁 환경, 사용자 페르소나, 검증 전략이 정의되었습니다.

## 주요 결정 사항

### 1. 시장 분석 결과
- **TAM**: 한국 개신교 교회 약 60,000개
- **SAM**: 디지털 관리 수요가 있는 중형교회 약 15,000개
- **SOM**: 초기 500개 → 3년 5,000개 교회 목표
- **경쟁사**: 디바인, 지저스온 등 기존 교회관리 솔루션 존재
- **차별점**: AI STT + 영적성장 도구 + 무료 홈페이지 빌더

### 2. 사용자 페르소나
- **목회자**: 행정 부담 경감, 교인 관리 효율화 필요
- **교회 행정 담당자**: 엑셀/수기 관리에서 디지털 전환 필요
- **성도**: 설교 노트, 성경 읽기 등 영적 성장 도구 필요
- **고령 교인**: 낮은 디지털 리터러시 → 극도로 단순한 UX 필수

### 3. GTM 전략
- ChurchThrive 컨설팅 기존 네트워크(127명 부트캠프 졸업생) 활용
- 파일럿 교회 3-5개로 MVP 검증
- 무료 티어로 진입장벽 제거 → STT 기능으로 유료 전환 유도
- 교단별/지역별 레퍼런스 교회 확보 전략

### 4. 검증 지표
| 지표 | Year 1 목표 |
|------|------------|
| 등록 교회 수 | 500 |
| 유료 전환 | 110 |
| MAU | 5,000 |
| NPS | 40+ |
