# 15분 빠른 시작 가이드

> **"오늘 15분 투자로, 평생 시간 절약"**
>
> 설명은 최소화하고, 실행만 따라하세요. 15분 후 첫 주보가 완성됩니다!

---

## ⏱️ 타임라인

```
0-3분:   Obsidian 다운로드 & 설치
3-5분:   첫 Vault 생성
5-8분:   폴더 구조 만들기
8-12분:  주보 템플릿 복사
12-15분: 첫 주보 작성 완료!
```

---

## 🚀 Step 1: Obsidian 다운로드 (3분)

### Windows 사용자

1. 브라우저에서 접속: https://obsidian.md
2. **"Download"** 버튼 클릭
3. **"Windows"** 선택
4. 다운로드된 파일 실행 (`Obsidian-X.X.X.exe`)
5. "설치" 클릭
6. 완료!

### Mac 사용자

1. 브라우저에서 접속: https://obsidian.md
2. **"Download"** 버튼 클릭
3. **"macOS"** 선택 (Intel/Apple Silicon 자동 선택)
4. 다운로드된 파일 실행 (`Obsidian-X.X.X.dmg`)
5. Obsidian 아이콘을 Applications 폴더로 드래그
6. Applications에서 Obsidian 실행
7. "열기" 클릭 (보안 경고 무시)

---

## 📁 Step 2: 첫 Vault 생성 (2분)

**Vault = 모든 문서가 저장되는 폴더**

1. Obsidian 실행
2. **"Create new vault"** 클릭

3. 설정:
   ```
   Vault name: 우리교회
   Location: 내 문서 폴더 (기본값)
   ```

4. **"Create"** 클릭

5. 잠시 후 비어있는 화면 나타남 ✅

**축하합니다! 첫 Vault가 생성되었습니다.**

---

## 🗂️ Step 3: 폴더 구조 만들기 (3분)

### 방법 1: 수동으로 만들기

1. 왼쪽 파일 목록에서 **우클릭**
2. **"New folder"** 선택
3. 폴더 이름 입력:
   ```
   100.교회운영
   ```
4. Enter

5. `100.교회운영` 폴더 안에 하위 폴더 3개 생성:
   - 우클릭 → New folder
   - 이름 입력: `110.주보`
   - 반복: `120.설교`, `130.회의록`

**완성된 구조**:
```
우리교회/
└── 100.교회운영/
    ├── 110.주보/
    ├── 120.설교/
    └── 130.회의록/
```

---

## 📝 Step 4: 주보 템플릿 준비 (2분)

1. 왼쪽에서 `110.주보` 폴더 클릭
2. 화면 가운데 **"Create new note"** 클릭
3. 파일 이름 입력:
   ```
   주보_템플릿
   ```
4. Enter

## 🎉 Step 5: 첫 주보 작성 (5분)

### 5-1. 템플릿 복사

1. 왼쪽에서 `주보_템플릿` 파일 **우클릭**
2. **"Duplicate"** 선택 (또는 `Ctrl+D`)
3. 새 파일 이름이 `주보_템플릿 1`로 생성됨
4. 이름 변경: `2025년 11월 17일 주보`

### 5-2. 내용 작성 (실제 정보 입력)

**수정할 부분들**:

1. **YAML (맨 위)**:
   ```yaml
   author:
     - "[[홍길동]]"  # 실제 담임목사 이름
   date created: 2025-11-17  # 오늘 날짜
   date: 2025-11-17  # 주일 날짜
   organization: "[[새소망교회]]"  # 실제 교회 이름
   ```

2. **설교 제목**:
   ```markdown
   설교: **하나님의 사랑** (요한복음 3:16)
   ```

3. **교회 소식**:
   ```markdown
   ### 감사와 기쁨의 소식
   - 김철수 집사님 장남 대학 합격 감사
   - 이영희 권사님 수술 무사히 마침 감사
   ```

4. **중보기도**:
   ```markdown
   ### 중보기도 제목
   1. 이번 주 전도 집회를 위해
   2. 새가족들의 신앙 성장을 위해
   3. 교회 건축을 위해
   ```

5. **헌금 및 연락처**:
   ```markdown
   **계좌번호**: 국민은행 123-456-789012 (새소망교회)

   ## 📞 연락처
   - **교회**: 02-1234-5678
   - **담임목사**: 010-1234-5678
   ```

### 5-3. 저장 및 확인

1. `Ctrl+S` (저장)
2. Reading View 확인 (오른쪽 상단 책 아이콘 클릭)
3. 주보가 깔끔하게 보이나요? ✅

**축하합니다! 첫 주보 완성!** 🎉

## ❓ 자주 묻는 질문 (FAQ)

### Q1: 파일이 어디에 저장되나요?
**A**: `내 문서/우리교회/` 폴더에 저장됩니다.
- Windows: `C:\Users\[사용자명]\Documents\우리교회\`
- Mac: `/Users/[사용자명]/Documents/우리교회/`

### Q2: 팀원들과 공유하려면?
**A**: 3가지 방법:
1. **OneDrive**: OneDrive 폴더에 Vault 생성
2. **Google Drive**: Google Drive 폴더에 Vault 생성
3. **USB**: Vault 폴더를 USB에 복사

**추천**: OneDrive (자동 동기화)

### Q3: 스마트폰에서도 볼 수 있나요?
**A**: 네!
1. 앱스토어/구글플레이에서 "Obsidian" 다운로드
2. 클라우드 폴더 (OneDrive/Google Drive) 연결
3. 동일한 Vault 열기

### Q4: 백업은 어떻게 하나요?
**A**: 자동 백업 2가지:
1. **클라우드**: OneDrive/Google Drive (추천)
2. **로컬**: Vault 폴더를 외장하드에 정기적으로 복사

### Q5: 무료인가요?
**A**:
- **개인 사용**: 완전 무료 ✅
- **비즈니스 (회사)**: $50/년 (교회는 개인 사용으로 간주)
- **클라우드 공식 동기화**: $10/월 (OneDrive 쓰면 불필요)

## 🆘 문제 해결 (Troubleshooting)

### 문제 1: Obsidian이 실행되지 않아요 (Windows)

**해결**:
1. 바이러스 백신 일시 중지
2. Obsidian 재설치
3. 관리자 권한으로 실행 (우클릭 → "관리자 권한으로 실행")

### 문제 2: 한글이 깨져요

**해결**:
1. Settings (톱니바퀴 아이콘)
2. Editor → Font
3. "Nanum Gothic" 또는 "맑은 고딕" 선택

### 문제 3: 파일이 사라졌어요!

**해결**:
1. 왼쪽 상단 돋보기 아이콘 클릭
2. 파일 이름 검색
3. 있다면: 다른 폴더에 있을 수 있음
4. 없다면: 삭제되었거나 저장 안 됨 → 백업 확인
