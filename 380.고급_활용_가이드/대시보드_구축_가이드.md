## 완성된 대시보드 예시

**파일 생성**: `교회_대시보드.md`

````markdown

## 📊 출석 현황

### 최근 4주 추이

```dataview
TABLE
  date AS "날짜",
  total_attendance AS "출석",
  (total_attendance - 200) AS "목표±"
FROM "100.교회운영/110.주보"
WHERE type = "bulletin"
  AND date >= date(today) - dur(28 days)
SORT date DESC
```

### 2025년 월평균

```dataview
TABLE
  dateformat(date, "yyyy-MM") AS "월",
  round(avg(rows.total_attendance), 0) AS "평균"
FROM "100.교회운영/110.주보"
WHERE type = "bulletin"
  AND date >= date("2025-01-01")
GROUP BY dateformat(date, "yyyy-MM")
SORT dateformat(date, "yyyy-MM") DESC
```

**분석**:
- 전월 대비: [직접 입력]
- 목표 달성: [직접 입력]

## 💰 재정 현황

### 주간 헌금

```dataview
TABLE
  date AS "날짜",
  offering AS "헌금",
  round(offering / 1000000, 1) + "M" AS "백만원"
FROM "100.교회운영/110.주보"
WHERE type = "bulletin"
  AND date >= date(today) - dur(56 days)
SORT date DESC
LIMIT 8
```

### 연간 목표 달성률

```dataview
TABLE WITHOUT ID
  "600,000,000원" AS "연간 목표",
  sum(offering) AS "현재 헌금",
  round((sum(offering) / 600000000.0) * 100, 1) + "%" AS "달성률"
FROM "100.교회운영/110.주보"
WHERE type = "bulletin"
  AND date >= date("2025-01-01")
```

## 🙏 심방 현황

### 이번 주 예정

```dataview
TABLE
  date AS "날짜",
  visitee AS "대상",
  visit_type AS "유형"
FROM "200.성도관리/220.심방기록"
WHERE type = "visit"
  AND date >= date(today)
  AND date <= date(today) + dur(7 days)
SORT date ASC
```

### 후속 심방 필요

```dataview
LIST
FROM "200.성도관리/220.심방기록"
WHERE type = "visit"
  AND follow_up = true
  AND follow_up_date <= date(today) + dur(14 days)
SORT follow_up_date ASC
```

### 월간 실적

```dataview
TABLE WITHOUT ID
  count(rows) AS "이번 달 심방"
FROM "200.성도관리/220.심방기록"
WHERE type = "visit"
  AND date >= date(today) - dur(30 days)
```
````

## Homepage 플러그인 활용

### 설정

1. Community Plugins → "Homepage" 설치
2. Settings → Homepage
3. "Open on startup": 체크
4. "Homepage file": `교회_대시보드.md` 선택

**효과**: Obsidian 열 때마다 대시보드가 자동으로!
